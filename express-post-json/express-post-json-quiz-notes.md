# express-post-json-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What does the `express.json()` middleware do and when would you need it?
  `express.json()` middleware parses JSON payload that are sent in the body of HTTP requests.
  It is needed when HTTP post, put or patch methods are used because client/user will be sending information to the server.
  The information sent to server is usually in JSON format.
